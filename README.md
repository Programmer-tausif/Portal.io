# 🌌 Portal.IO


A sleek, responsive, and dynamic web application featuring an asynchronous Email Authentication system (Login/Signup). Built with a premium, high-contrast dark theme on the frontend and a secure PHP backend using PDO.

## ✨ Key Features
* **Modern Dark UI:** Deeply saturated dark tones accented with vibrant neon highlights.
* **Asynchronous Auth:** Login and registration forms process instantly using JavaScript's Fetch API (no annoying page refreshes).
* **Secure Backend:** PHP 8+ handling native user sessions and password safety via `PASSWORD_BCRYPT` hashing.
* **Prepared SQL Structure:** Preventative database architecture using PDO statements.

## 🛠️ Tech Stack
* **Frontend:** Semantic HTML5, Custom CSS3 Variables, Modern JavaScript (ES6+)
* **Backend:** PHP 8+ (Session Management, JSON API)
* **Database:** MySQL / XAMPP

## 🚀 Local Setup Instructions
1. Clone or download this repository into your XAMPP `htdocs` folder.
2. Open **phpMyAdmin** and import the provided `database.sql` file.
3. Rename `db.example.php` to `db.php` and configure your local credentials if needed.
4. Turn on Apache and MySQL in your XAMPP Control Panel.
5. Open your browser and navigate to `http://localhost/YOUR_FOLDER_NAME/`.
